前言

如果你也想自己写属于自己的 api 可以看这个篇文章

  • laravel 官方文档 10
  • 中文版 9

安装必须工具

  • 首先先安装 composer https://getcomposer.org/ 全程下一步
  • 安装 php，数据库等 个人使用的是小皮这个方便可以一键安装
    • 官方链接 https://www.xp.cn/
    • https://public.xp.cn/upgrades/phpStudy_64.zip也可以点这个链接直接下载也是全程下一步

安装 laravel

安装完就可以用 vscode 随便打开一个文件夹然后终端里面打这句 example-app 是你的项目名称 **打开你的小皮启动一下,更改一下目录打开网址就行，我这里使用的是数据库 8 和 php8**


选到你创建的目录下的 public 就可以了，去 posman 上测试接口的话要在你复制的网址后面加上 index.php 这样就创建好了一个 php 网站了

  • 路由页面在resources/views下面创建

    • 这几种是常用的，想知道更多请查看文档

    • post
    • put
    • patch
    • delete
    • …等等
  • web.php 这个是写网页路由
    • 基本路由
    • 默认路由文件 利用控制器UserController件来执行index()函数
    • 视图路由 ‘/welcome’ 展示views下面的welcome页面
    • 路由名前缀 prefix(‘xxx’) 访问时需要在路由前面增加xxx

数据库

  • 检索表单的内容
    • DB::table(‘users’)->get();
  • 检索单行或单列
    • DB::table(‘users’)->where(‘name’, ‘John’)->first();
  • 通过id获取单行数据
    • DB::table(‘users’)->find(3);
  • 获取某一列的值
    • DB::table(‘users’)->pluck(‘title’);
  • select语句
    • DB::table(‘users’)->select(‘name’)->get();
  • join语句
    • DB::table(‘users’)->join(‘contacts’, ‘users.id’, ‘=’, ‘contacts.user_id’)->select(‘users.*’, ‘contacts.phone’, ‘orders.price’)->get();
    • 在users表和contacts表 查询相等的内容 输出
  • where语句
    • DB::table(‘users’)->where(‘votes’, 100)->get();
  • 插入语句
    • DB::table(‘users’)->insert([内容]);
  • update语句
    • DB::table(‘users’)->where(‘id’, 1)->update([‘votes’ => 1]);
  • 更新或新增 (查询到就更改，查不到就新增数据)
    • DB::table(‘users’)
      ->updateOrInsert([‘email’ => ‘john@example.com’, ‘name’ => ‘John’],[‘votes’ => ‘2’]);
  • 删除语句
    • DB::table(‘users’)->delete();
  • 重置 (从表中删除所有记录并将自动递增 ID 重置为零)
    • DB::table(‘users’)->truncate();

数据库的迁移

  • 生成迁移
    • php artisan make:migration create_flights_table
  • create_flights_table结构
  • 执行迁移
    • php artisan migrate
  • 重置数据库，并运行所有的 seeds…
    • php artisan migrate:refresh —seed
  • 删除所有表然后执行迁移
    • php artisan migrate:fresh —seed

数据填充

  • 新建Seeders
    • php artisan make:seeder UserSeeder
  • 编写 可以用for循环填充也可以用模型工厂
  • 模型工厂
    • database/seeders/DatabaseSeeder.php
    • use App\Models\Users;
    • users::factory(50)->create();

Eloquent ORM (模型)

  • 生成模型
    • php artisan make:model Flight 或 Flights
    • php artisan make:model Flight —migration 或 Flights —migration 同时生成迁移文件
      • 对应的生成迁移的数据库里要有 flights 这个表单 如果没用就要在 model 里面设定
    • protected $table = ‘xxx’; 数据库对应的表名
      • 如果不是id为主键的话要设定一个相对应的主键
      • protected $primaryKey = ‘xxxid’; 对应的键名设定为主键
